# Top Seekers

Top Seekers is a simple and visually appealing website project built using HTML, CSS, and images.

## Features

- Clean UI design
- Multiple image sections
- Simple static website structure
- Beginner-friendly project

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in browser
